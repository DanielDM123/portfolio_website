Fun With Rotoscoping Project

PROJECTS

This project involved implementing three core computer graphic and animation techniques (rotoscoping, image wrapping, and keying) to create a video that was at least 30 seconds long.

Rotoscoping

Created animation by tracing live-action footage of myself frame-by-frame.

Creating a program that stored points on the screen into an XML file.

Once the program read the incoming frame, it draw out and connect the points read from the XML file.

If the program notice there was only one point on the current frame, it would instead draw a PNG of a bird where the point was at.

Image Warping

Manipulated the PNG of the bird by translating the pixels of the head left and right.

Achieved this by determining what y-level separated the neck and body of the bird.

Then, when drawing the bird image, every pixel above the threshold would move either left or right in a linear fashion by utilizing linear interpolation.

Chromakeying

Implemented my own green screen background replacement functionality.

The program would generates an alpha matte for each frame by applying Vlahos equations, a method used to differentiate the green screen background from the foreground.

Afterwards, the program would compare it against a predefined garbage mask to precisely isolate the intended foreground from extraneous background elements.
